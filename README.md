## Whatsapp Bot Koentji Surga V.1

### Main Features

| Features         | Available |
| ---------------- | --------- |
| Generator Hadist | ✅        |
| Jadwal Sholat    | ✅        |

#### Step Installation

- clone repositori ini
- jalankan

```
npm install
```

- untuk running program silahkan ketik

```
npm start
```
